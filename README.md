Aplikacja ta umożliwia tworzenie grafu oraz liczenie najkrótszej ścieżki za pomocą algorytmu dijkstry. Aplikacja jest napisana w pythonie i wykorzystuje bibioteki tkinter, matplotlib i networkx.
